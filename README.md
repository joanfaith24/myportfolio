## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/joanfaith24/myportfolio.git

# Navigate into the project
cd myportfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open **http://localhost:5173** in your browser.
